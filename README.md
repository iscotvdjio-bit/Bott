✨ About

ViBE Point Bot adalah bot Discord dengan sistem ekonomi modern yang memungkinkan user mendapatkan point, bersaing dalam leaderboard, dan melakukan donasi antar user.

Dirancang untuk membuat server kamu lebih aktif, kompetitif, dan terlihat premium.

---

🔥 Features

- 💰 Point system (auto earn dari aktivitas)
- 🏆 Ranking & leaderboard
- 🎁 Donate system (monthly top donator)
- 📊 Statistik user
- ⚙️ Fully customizable
- 🛡️ Anti abuse protection

---

🧠 How It Works

«Sistem berjalan otomatis berdasarkan aktivitas user.»

- Chat → dapat point
- Donate → boost rank
- Leaderboard → kompetisi

---

🎮 Commands

Command| Description
"/point"| Check point kamu
"/rank"| Lihat rank
"/top"| Leaderboard server
"/donate"| Donasi ke user
"/donate-top"| Top donator bulan ini
